# 🦖 Frosky AI Assistant

**Your personal AI assistant for cryptocurrency, investing, and learning!**

## Features
- 🦖 Real-time Bitcoin price tracking
- 🚀 Conversation memory
- 💾 Chat history persistence
- 🔥 Crypto, investing & education specialist
- ⚡ Live API integrations

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- AI: Groq API + Llama 3.1 8B
- Data: CoinGecko API
- Storage: localStorage (browser)

## Getting Started

### 1. Deploy to GitHub Pages
This site will automatically deploy to GitHub Pages when pushed to `main` branch.

### 2. Get API Key
- Sign up at: https://console.groq.com/login
- Create API key (starts with `gsk_`)
- Paste in app when you visit

## How It Works

1. **Chat Interface** - Simple, clean web UI
2. **Contextual Memory** - Remembers last 20 messages
3. **Live Data** - Real-time Bitcoin & crypto prices
4. **Persona** - Froskybtc (friendly, educational, Ghana-based)

## Security
- API keys stored only in your browser (localStorage)
- No backend server required
- All data stays on your device
- Groq API calls go directly from browser

## Customization
Want to modify? Edit `index.html` with your custom features!
