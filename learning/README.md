# Learning Python & Web Development

## Quick Start

### Step 1: Get a Code Editor
- **Best:** [VS Code](https://code.visualstudio.com/download) (free, recommended)
- **Alternative:** [Sublime Text](https://www.sublimetext.com/), [Notepad++](https://notepad-plus-plus.org/)

### Step 2: Follow the Lessons
- `lesson-01-html-basics.md` - HTML fundamentals
- `lesson-01-html-basics-part2.md` - Building sections
- `index.html` - Your first complete website

### Step 3: Practice Today
1. Open the `index.html` file
2. Open it in your browser
3. **Change the text** - write your own name, add your own services
4. **Try different tags** - add more `<p>`, `<h2>`, etc.
5. **Break it, fix it** - learn by making mistakes
