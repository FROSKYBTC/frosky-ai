# GitHub Setup for Frosky Construction Website

## Step-by-Step (Do this NOW):

### 1. OPEN YOUR TERMINAL FIRST:

**Windows:**
- Press Windows key, type `cmd`
- Click **Command Prompt**

**Mac:**
- Press Command + Space
- Type `terminal`, press Enter

**Linux:**
- Press Ctrl + Alt + T

---

### 2. Navigate to the workspace folder

Type in terminal (press Enter after):

```bash
cd ~/.openclaw/workspace
```

You should see your files pop up if you type:

```bash
ls
```

(You should see `my-website.html`)

---

### 3. Create Repository on GitHub
- Go to https://github.com/new
- **Repository name:** `frosky-construction`
- **Description:** "Frosky Construction Company - Building Dreams"
- **Public/Private:** Your choice
- ❌ **Uncheck** "Add README"
- ✅ Click **Create repository**

### 4. Get Your Git Commands
After creating, you'll see something like:
```
git remote add origin https://github.com/YOUR-USERNAME/frosky-construction.git
git branch -M main
git push -u origin main
```

### 5. Copy PASTE those commands into YOUR TERMINAL!

Replace `YOUR-USERNAME` with your actual GitHub username.

Press Enter after each line.

---

### 6. Enable GitHub Pages

1. Go to your repo: `https://github.com/YOUR-USERNAME/frosky-construction`
2. Click **Settings** → **Pages** (left sidebar)
3. Select **main** branch, **/ (root)** folder
4. Click **Save**

**DONE!** Your site will be live at:
`https://YOUR-USERNAME.github.io/frosky-construction/`

---
