# Practice Today: Your First HTML Edit

## What You'll Do:

### 1. See it work (5 min)
**Open:** `~/.openclaw/workspace/learn/simple-site.html`

In File Explorer:
- Windows: Press `Windows + E` → Navigate to folder → Double-click file
- Mac: Open Finder → Go to `~/.openclaw/workspace/learn/` → Double-click

**OR** in terminal (if you have it open):
```bash
cd ~/.openclaw/workspace/learn/
open simple-site.html     # Mac
# OR
start simple-site.html    # Windows
# OR
xdg-open simple-site.html # Linux
```

**Expected result:** Browser opens with your simple construction site!

### 2. First Edit: Change the Title (5 min)
1. Right-click `simple-site.html` → Open with **Notepad** (or VS Code)
2. Find this line:
   ```html
   <h1>Welcome to Frosky Construction</h1>
   ```
3. Change it to:
   ```html
   <h1>Welcome to Lawrence's Construction</h1>
   ```
4. Save (Ctrl+S / Cmd+S)
5. Refresh browser
6. **See your name now appear!**

### 3. Add Something (10 min)
Try adding a new service:
```html
<li>Custom Homes</li>
```

Or change your contact info:
```html
<p>📞 +233 000 000 000</p>
```

### 4. Try Breaking It! (5 min)
- Change one closing tag to be wrong: `</h2>` → `</h3>`
- See what breaks
- Fix it
- **Learning by making mistakes!**

### 5. Ask Questions!
- What does `<h1>` mean?
- Why use `<ul>` vs just typing bullet points?
- How do I add a photo?

**I'm here to answer ANYTHING!**
