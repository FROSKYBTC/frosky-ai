# STEP-BY-STEP: Learn HTML by Editing Your Site

## What you'll do today (2-3 hours):

### Part 1: See it work (5 min)
1. Open File Explorer / Finder
2. Go to: `~/.openclaw/workspace/learn/`
3. Double-click `simple-site.html`
4. **It opens in browser!** See your site!

### Part 2: Edit it! (15 min)
1. Open `simple-site.html` in VS Code (or Notepad)
2. Find this line:
   ```html
   <h1>Welcome to Frosky Construction</h1>
   ```
3. Change `Frosky Construction` to YOUR name
4. **Save** (Ctrl+S / Cmd+S)
5. **Refresh browser**
6. **Look at your changes!**

### Part 3: Add more content (45 min)
Change the text:
- Add your city
- Add your phone number
- Add new services
- Try adding an `<img>` tag for photos

### Part 4: Ask questions! (remaining time)
- What does that tag do?
- How do I change colors?
- How do I add a button?

**I'm here to explain EVERYTHING!**
